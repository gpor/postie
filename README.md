# Postie
### Simple API checker



built on [nuxt 3](https://v3.nuxtjs.org).

## Install

Make sure to install the dependencies:

```bash
npm i
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

## CORS for your API server

[enable-cors.org](https://enable-cors.org/server.html)
